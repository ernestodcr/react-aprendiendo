# 02 Componentes Básicos

Segundo proyecto de React donde practico la creación y uso de componentes.

## Demo
https://react-aprendiendo-7dfx.vercel.app/

## Objetivos
- Crear componentes funcionales
- Separar la interfaz en piezas (Header, Main, Footer)
- Importar y exportar componentes
- Estructurar una aplicación básica en React

- App.jsx

## Tecnologías
